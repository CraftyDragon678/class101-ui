## onChagne

<Playground>
  {() => {
    class Example extends React.Component {
      constructor(props) {
        super(props);
        this.state = {
          value: '',
        };
      }
      render() {
        return (
          <Select
            value={this.state.value}
            onChange={e => this.setState({ value: e.target.value })}
            options={['Apple', 'Banana', 'Candy']}
          />
        );
      }
    }
    return <Example />;
  }}
</Playground>

> `inline:true`를 사용할 경우에 `fill` 값을 변경해도 형태가 바뀌지 않습니다.
