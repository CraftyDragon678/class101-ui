# Input

문자열을 입력받는 컴포넌트입니다.

## Props

<PropsTable of={Input} />

## Basic usage

<Playground>
  <Input placeholder="입력해주세요." />
  <br />
  <Input size="sm" placeholder="입력해주세요." />
</Playground>

## Size

기본 사이즈는 `md`입니다.

> class101 제품에서 `sm` 사이즈는 폼이 많은 관리자 단에서만 사용해주세요.

<Playground>
  <Input size={InputSize.md} />
  <br />
  <Input size={InputSize.sm} />
  <br />
  <Input size={InputSize.md} fill={false} />
  <br />
  <Input size={InputSize.sm} fill={false} />
</Playground>

## Inline input

<Playground>
  <Input inline placeholder="입력해주세요." />
  <Input inline placeholder="입력해주세요." />
</Playground>

## Fill

블록 영역에서 꽉차게 사용할 지를 결정합니다.

> `inline:true`를 사용할 경우에 `fill` 값을 변경해도 형태가 바뀌지 않습니다.

<Playground>
  <Input placeholder="입력해주세요." fill={true}/>
  <Input placeholder="입력해주세요." fill={false}/>

  <Input placeholder="입력해주세요." inline={true} fill={true}/>
  <Input placeholder="입력해주세요." inline={true} fill={false}/>
</Playground>
